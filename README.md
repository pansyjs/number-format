<div align="center">
  Format numbers as a text. 将数字格式化为文本。
</div>

## ✨ 特性

- 📦 开箱即用
- 💝 无任何第三方依赖
- 💻 使用 TypeScript 编写

## 🏗 安装

```sh
# npm install
$ npm install @pansy/number-format --save

# yarn install
$ yarn add @pansy/number-format
```

## 🔨 使用

```ts
import numberFormat from '@pansy/number-format';
```
